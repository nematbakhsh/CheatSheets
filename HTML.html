Global Attributes

accesskey=STRING character Specifies a shortcut key to activate/focus an element class=STRING classname(s) Specifies classname(s) for an element (usually refers to a class in a style sheet) contenteditable=STRING true|false Specifies whether the content is editable or not data-*=STRING value creates custom data attributes dir=STRING ltr|rtl|auto Specifies the text direction draggable=STRING true|false|auto Specifies whether an element is draggable or not (often used in drag and drop operations) hidden Hides an element as if it doesn't exists. It is a boolean attribute id=STRING id Specifies a unique id (usually used by javascript to select the element lang=STRING language-code Specifies the language of the element's content Language Codes spellcheck=STRING true|false Specifies whether the element's content is to have its spelling and grammar checked or not style=STRING style(s) Specifies inline style(s) (i.e. CSS properties and values) seperated by ';'. Override any style set globally tabindex=STRING tab-order Specifies the tabbing order title=STRING Specifies extra information about an element (This information is often shown as a tooltip text when mouse moves over.) =STRING translate=STRING yes|no Specifies whether the content of an element should be translated or not
