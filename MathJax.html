MathJax

Beautiful and accessible math in all browsers

Installation

Snippet integration using a CDN

Configuration
Inside your html file
<script>
MathJax = {
  tex: {
    inlineMath: [['$', '$'], ['\\(', '\\)']]
  }
};
</script>
<!-- It is important that the configuration above comes before the script that loads MathJax component-->
<script id="MathJax-script" async
  src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-chtml.js">
</script>
