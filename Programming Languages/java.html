Java

Java is a high-level, class-based, object-oriented programming language developed by Oracle Corporation.

Installation

  • Download Java SE Development Kit (JDK) based on you operating system.
  • Install JDK
  • (Optional) Install a specific code editor
    Most Popular:
    • IntelliJ
    • NetBeans
    • Eclipse

General Properties

  • Java is case sensitive.
  • Statement terminator is semicolon (;)
  • line comments by double slashes (//)

Hello World

package com.examples;

public class HelloWorld {
  public static void main(String[] args) {
  	System.out.println("Hello World");
  }
}
