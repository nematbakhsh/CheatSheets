FileReader Interface

This object asynchronously reads the contents of files (or raw data buffers) from the users's computer, using File or Blob bjects.

General Usage

Inside .js file or script tag
// constructor
const fileReader = new FileReader()

// handle load and error events
//
// upon load the result and error are in fileReader.result and fileReader.error properties
fileReader.onload = function (event) {
    // actions after load completes
    // access result either as event.target.result or fileReader.result
    console.log(event.target.result)
}

fileReader.onerror = function(event) {
	// handle errors
	alert("Load Failed!\n" + fileReader.error)
}

// start reading the contents of the specified Blob
fileReader.readAsText(OBJECT File/Blob)
// this will trigger 'loadstart', 'progress', 'loadend', 'load'/'error' events sequentially which can be handled by either an event handler or by addEventListener() method.
