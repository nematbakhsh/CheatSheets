Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development.

Installation

  • Download from Boostrap Website
  • NPM :
    npm install bootstrap
    npm i bootstrap-icons

    const bootstrap = require('bootstrap') or import bootstrap from 'bootstrap' will load all of Bootstrap’s plugins onto a bootstrap object. The bootstrap module itself exports all of our plugins. You can manually load Bootstrap’s plugins individually by loading the /js/dist/*.js files under the package’s top-level directory. 

    // How to bring in with Express.js
app.use("/bootstrap", express.static(path.join(__dirname, "node_modules/bootstrap/dist/css"))); // <- This will use the contents of 'bootstrap/dist/css' which is placed in your node_modules folder as if it is in your '/styles/css' directory.
app.use("/bootstrap", express.static(path.join(__dirname, "node_modules/bootstrap/dist/js")));
app.use("/bootstrap", express.static(path.join(__dirname, "node_modules/bootstrap-icons/font")));
app.use("/bootstrap", express.static(path.join(__dirname, "node_modules/bootstrap-icons")));

Layout

Breakpoints (grid tiers)

xs or <no-class-name> X-Small (< 576px) sm Small (≥ 576px) md Medium (≥ 768px) lg Large (≥ 992px) xl X-Large (≥ 1200px) xxl XX-Large (≥ 1400px)

Containers

container wsets a max-width at each responsive breakpoint (a little less than the breakpoint width except X-Small which is width : 100%) container-fluid is width : 100% at all breakpoints container-{breakpoint} is width : 100% until the specified breakpoint

Grid

Usage

<div class="container">
    <div class="row">
        <div class="col">
            Column
        </div>
        <div class="col">
            Column
        </div>
    </div>
</div>
Column
Column
row sets a row (flex-container) row-cols-<breakpoint>-<NUMBER> sets a row with specified number of columns at sepecified breakpoint (<breakpoint> omitted means for all width) col auto width specified column col--<breakpoint>-<NUMBER | auto> column with specified width at specified breakpoint (<breakpoint> omitted means for all width, <breakpoint> and <NUMBER> omitted means column width is automatically specified due to the number of children) (the auto option will size column based on the natural width of its content.
