Usage

Font

Generic font families

  • Serif have small strokes at the edges of each letter. (Formal/Elegant look)
  • Sans-serif have clean lines (no small strokes). (Modern/Simple/Minimalistic look)
  • Monospace all the letters have the same fixed width. (Mechanical/Old/Code look)
  • Cursive imitate human handwriting. (Friendly/Casual look)
  • Fantasy decorative/playful fonts. (Decorations/Games)

Tips

Properties that accept length but not percentages: text-shadow, border, letter-spacing

Properties that accept percentages but not length: text-size-adjust

Good Practice to set pading and margin to zero for all elements first

To center elements horizontally (along y-axis) 

// for texts use text-align:center
.center-horizontally {
    display : block;
    margin-left : auto;
    margin-right : auto;
    width : ? // note that if width is not set or set to 100% then horizontal aligning has no effect
}

To center elements verically (along x-axis) 

// Method 1
.center-vertically {
    padding-top : LENGTH l
    padding-bottom : LENGTH l
}
// Method 2
.centering-vertically-parent {
	position: relative;
}
.centering-vertically-element {
	margin: 0;
    position: absolute;
    top: 50%;
    -ms-transform: translateY(-50%);
    transform: translateY(-50%);
}

To center horizontally and vertically 

// Method 1
.centering-parent {
	position: relative;
}
.centering-element {
	margin: 0;
    position: absolute;
    top: 50%;
    left: 50%;
    -ms-transform: translate(-50%, -50%);
    transform: translate(-50%, -50%);
}
// Method 2
.center {
    display: flex;
    justify-content: center;
    align-items: center;
}

Technicalities

Margin Collapsing

Margin collapsing occurs in these three basic cases:

  • Adjacent siblings
  • No content separating parent and descendants
  • Empty blocks
Technics to prevent this:
  • Float either of div elements
  • Change either of your elements' disply to inline-blocks
  • Set overflow of parent to auto (or any value other than visible)
