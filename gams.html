Commands in order of Appearance

$Title <Title> Title of the Problem comes afterwards * <Comment> Single line comment $Ontext <Comments> $Offtext Block of comments

Sets

set[s] set_name ["text"] [/element [text] {,element [text]} /]

{,set_name ["text"] [/element [text] {,element [text]} /] } ;

Note that each element in a set must be separated from other elements by a comma or by an end-of-line, and each element is separated from any associated text by a blank.

Data Entry


scalar[s] scalar_name [text] [/numerical_value/]
    {scalar_name [text] [/numerical_value/]} ;
Defines a 0-dimensional parameter. Note that more than one scalar may be declared in one scalar statement. The entries have to be separated by commas or by end of line. scalar_name = numerical_value initializes the scalar with an assignment statement 

parameter[s] param_name[(index_list)] [text] [/ element [=] numerical_value
                                              {,element [=] numerical_value} /]
           {,param_name[(index_list)] [text] [/ element [=] numerical_value
                                              {,element [=] numerical_value} /]} ;
Element is an element of the defining set or - if there is more than one defining set - a combination of the elements of the defining sets. The default value of a parameter is 0. Several parameters may be declared in one parameter statement. 

table table_name[(index_list)]  [text] [EOL
                element               { element }     EOL
  element    numerical_value       { numerical_value} EOL
 {element    numerical_value       { numerical_value} EOL}] ;
Element is an element of one of the driving sets.

Variables


[var_type] variable[s] var_name [(index_list)] [text] [/var_data/] 
    {, var_name [(index_list)] [text] [/var_data/]}
var_type can be free(default), positive/nonnegative, negative, binary, integer, etc.

Equations


Equation[s] eqn_name [(index_list)] [explanatory text] [/eqn_data/]
    {, eqn_name [(index_list)] [explanatory text] [/eqn_data/]} ;
Declare equation eqn_name(index_list)[$logical_condition(s)].. expression eqn_type expression ; Define equation. eqn_type can be =e=, =l=, =g= , etc.
