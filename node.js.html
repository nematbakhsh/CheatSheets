Installation

Install from nodejs.org

NPM

node -v | node --version returns node.js version npm -v | npm --version returns npm version npm init initializes an app by creating a package.json file

packaje.json content

"name" : STRING "version" : STRING "description" : STRING "keywords" : Array array-of-strings "homepage" : STRING "bugs" : { "url" : STRING, "email" : STRING} "main" : STRING "scripts" : OBJECT "author" : STRING "license" : STRING "dependencies" : OBJECT "devDependencies" : OBJECT "repository" : { "type" : STRING, "url" : STRING "private" : true npm will refuse to publish private apps

Path module

const path = require('path') calls the module path.basename(STRING filepath) returns only the base filename with extension path.dirname(STRING filepath) returns only the directory path.extname(STRING filepath) return file extension with dot path.parse(STRING filepath) rturns path object (an object with keys: root, dir, base, ext and name) path.join(...paths) concatenates paths path.sep Provides the platform-specific path segment separator: '\' for windows and '/' for POSIX path.resolve([...paths]) The given sequence of paths is processed from right to left, with each subsequent path prepended until an absolute path is constructed

Tip: path.resolve(__dirname, '..') returns the parent directory
