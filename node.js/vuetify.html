Vue.js Vuetify Plugin

A Material Design Framework for Vue.js

Installation

vue add vuetify // (on windows) winpty vue.cmd vue add vuetify

Above command will change some on the files and adds the vuetify.js in the plugin folder

Useful Props

app
in the v-toolbar makes it fixed
dark/light
boolean prop
height="<value>"
left, right
boolean - pulls the object to left or right
router
makes it like an anchor tag - then use to prop
to
somehow like href attribute - bind it if the address is coming from a data variable
fluid, block
outline
shows the outline of tag element

It seems that props override classes

Setup

Inside vuetify.js
import Vue from 'vue';
import Vuetify from 'vuetify/lib/framework';

Vue.use(Vuetify);

export default new Vuetify({
  // optional settings
  icons: {
    iconfont: 'mdiSvg', // 'mdi' || 'mdiSvg' || 'md' || 'fa' || 'fa4' || 'faSvg'
  },
  theme: {
    themes: {
      light: {
        primary: '#3f51b5',
        secondary: colors.grey.darken1,
        accent: colors.shades.black,
        error: colors.red.accent3,
      },
      dark: {
        primary: colors.blue.lighten3,
      },
    },
  },
})
